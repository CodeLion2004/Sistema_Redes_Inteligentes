Recomendaciones operativas

Relación con el patrón Command

Interfaz ComandoRed: define el contrato con los métodos ejecutar(), deshacer() y getNombre(), permitiendo que el invocador trate todos los comandos de forma uniforme.
Comandos concretos: ComandoActualizarLimiteConsumo, ComandoCambiarFuenteEnergia y ComandoCambiarNivelAlerta encapsulan cada operación junto con el estado previo necesario para deshacerla.
InvocadorComandosRed: actúa como el Invoker del patrón; mantiene una pila (Deque) del historial de comandos ejecutados y expone ejecutar(), deshacerUltimo() y obtenerHistorial().
Soporte Undo: cada comando guarda el valor anterior antes de aplicar el cambio, lo que permite revertir cualquier modificación de configuración de red de forma segura e independiente.
